# Chilescar 

Aplicación web desarrollada con React que permite visualizar información de pilotos y datos relacionados mediante consumo de API REST, mostrando información en tiempo real.

## Tecnologías utilizadas
- React
- JavaScript
- HTML / CSS
- API REST
- (Si usaste backend: Spring Boot / Node.js)

- ## Cómo ejecutar el proyecto

1. Clonar repositorio:
git clone https://github.com/SaltEagle/Chilescar.git

2. Entrar a la carpeta:
cd Chilescar

3. Instalar dependencias:
npm install

4. Ejecutar:
npm start


## Funcionalidades
- Consumo de API REST para obtener datos dinámicos
- Renderizado de información en tiempo real
- Componentes reutilizables en React
- Manejo de estado

## Aprendizajes
- Consumo de APIs REST en aplicaciones frontend
- Manejo de estado en React
- Estructuración de componentes reutilizables
- Integración frontend-backend
